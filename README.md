# ⌨️ Typing Game

**Typing Game** is a fast-paced browser-based application built with **HTML**, **CSS**, and **JavaScript**. Designed to improve typing speed and accuracy, this game challenges users to type dynamically generated words within a time limit, offering real-time feedback and a responsive user experience.

This project showcases my skills in **frontend web development**, **DOM manipulation**, and **JavaScript event handling**—all wrapped in a sleek and interactive interface.

---

## 🚀 Project Goals

- Build an engaging, interactive typing speed game using vanilla web technologies
- Practice real-time user input processing with keyboard events
- Design responsive UI components using semantic HTML and modern CSS
- Write modular, maintainable JavaScript code for real-time interactivity

---

## ✨ Features

- ⚡ **Real-time Typing Detection**: Instant feedback as you type each letter
- ⏱ **Countdown Timer**: Race against the clock to test your speed
- 📈 **Live Score Tracking**: Displays correct word count and typing accuracy
- 🔄 **Auto Word Generation**: Random words appear as you progress
- 🎯 **Difficulty Scaling**: Optional logic to increase difficulty with time (planned)
- 📱 **Responsive Design**: Mobile-friendly layout and fluid UI

---

## 📦 Technologies Used

- **HTML5** for structure and semantics
- **CSS3** for styling and responsive design
- **JavaScript (ES6+)** for logic, event handling, and DOM updates
- **Keyboard Events**: `keydown`, `keyup`, `input` to track user input
- **Dynamic DOM Manipulation** for word rendering and feedback

- Real-time game logic & state management
